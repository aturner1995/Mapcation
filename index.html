Airline API Integration


Mapcation integrates with a flight finder API, displaying real-time flight data from multiple airlines, sorting and filtering results by price, duration, and airline.

Trip Advisor API Integration


Mapcation showcases top-rated tourist attractions and activities with Trip Advisor API, allowing users to browse through photos, reviews, and ratings to plan their trip.

Open Street Maps API Integration


Mapcation displays the location of the chosen destination and top-rated tourist attractions with Open Street Maps API, letting users zoom in and out on the map.

YouTube API Integration


Mapcation features YouTube API integration, allowing users to access videos about their chosen travel destination. Users can browse a selection of videos and watch them within the app.
